# Сторонние источники и лицензии

## Карта и геоданные

- Yandex Maps JavaScript API 3.0 — используется по условиям сервиса API Яндекс Карт.
- `@yandex/ymaps3-types` — Apache-2.0.

Логотип, копирайты и ссылка «Открыть в Яндекс Картах» выводятся штатными компонентами API и не скрываются интерфейсом сайта.

## Библиотеки проекта

- Turf.js — MIT.
- Vite — MIT.
- TypeScript — Apache-2.0.
- Sharp — Apache-2.0.
- exifr — MIT.
- tsx — MIT.

Полный перечень транзитивных зависимостей и их версии зафиксирован в `package-lock.json`.

## Материалы истории

- Фотографии путешествия находятся в `assets/photos/` и принадлежат владельцу проекта.
- Ассет автодома с видом сверху создан специально для проекта без сторонней фотографии и хранится в `assets/generated/camper-top-final.png`.
- Звуковой фон генерируется локально Web Audio API; внешние музыкальные и звуковые файлы не используются.
